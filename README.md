# 始発スタジオ｜楽曲提供サイト

シーンで選べる著作権フリーBGMの提供ページ。試聴・YouTube視聴・ダウンロードに対応。

- ひかり（朝・通学）／しおん（夜・ゲーム）のテーマ切替
- 各曲：その場で試聴／YouTubeのタイムスタンプ位置から再生／mp3ダウンロード
- 収録基準：高評価3以上 × タイムスタンプ導入後の動画（Rain-Clear Hymn・ゲームメニュー）

YouTubeチャンネル: https://www.youtube.com/@shihatsu_bgm

## 構成
- `index.html` … サイト本体（1ファイル完結・依存なし）
- `audio/` … 配布mp3
- `img/` … カバー画像

## 公開
GitHub Pages（このリポジトリの Settings → Pages → Branch: main / root）で公開。
